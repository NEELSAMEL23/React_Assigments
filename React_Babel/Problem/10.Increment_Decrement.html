L0 - Counter with Step Control

Problem Statement:

reate a React functional component that includes:

  1. A counter starting at 0.
  2. "Increase" and "Decrease" buttons to update the counter.
  3. An input field to specify a step value. By default, the step value is 1.
  4. Buttons should increment or decrement the counter based on the step value entered by the user.

For example:

  • If the step value is 3 and the user clicks "Increase," the counter should increase by 3.

Ensure the counter cannot go below 0.
