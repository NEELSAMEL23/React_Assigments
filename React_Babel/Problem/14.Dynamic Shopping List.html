L1 - Dynamic Shopping List with Quantities

Problem Statement:

Create a shopping list component with the following features:

  1. Users can add items with a name and quantity (e.g., "Apples - 2 kg") through an input field.
  2. Display the list dynamically as items are added.
  3. Include a "Remove" button to delete items from the list.
  4. Add a "Clear All" button to remove all items from the list at once.

Bonus: Add validation to prevent adding empty items or quantities less than 1.
