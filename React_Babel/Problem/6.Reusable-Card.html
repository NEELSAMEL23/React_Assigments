L0 - Building a Reusable Card Component

Problem Statement:

Develop a reusable Card component that can display a title and content. Use the children prop to allow dynamic content inside the card.

Requirements:

  1. The Card component should accept the following props:
    • title (string): The title of the card.
    • children: Dynamic content for the card's body.
  2. Create a parent component that renders at least three Card components with different titles and contents.
