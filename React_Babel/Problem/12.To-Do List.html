L1 - Debugging a Multi-Feature To-Do List

Problem Statement:

You are given a buggy React To-Do List component with the following issues:

  1. The "Add Task" button sometimes adds empty tasks.
  2. Completed tasks are not visually distinguishable (no strike-through).
  3. The "Delete Task" button deletes the wrong task occasionally.

Your task is to debug and implement the following features:

  1. Ensure only non-empty tasks can be added.
  2. Tasks marked as completed should have a visible strike-through.
  3. Ensure the "Delete Task" button always removes the correct task.

Write a short note explaining how you fixed each bug.
