L2 - Dynamic Blog Page with Featured Highlight

Problem Statement:

Create a dynamic blog page that displays a list of blog posts with the following features:

Requirements:

  1. Each blog post should include
    • A title (string).
    • Content (string).
    • A boolean isFeatured indicating if the blog is featured.
  2. Highlight featured blogs with a different background color (e.g., light yellow).
  3. Use map() to dynamically render all blogs from a provided array.

Example Array:

const blogs = [
{ title: "React Basics", content: "Learn the basics of React.", isFeatured: true },
{ title: "Advanced React", content: "Delve deeper into React.", isFeatured: false },
{ title: "React Performance Tips", content: "Optimize your React apps.", isFeatured: true },
];
