L0 - Toggle Button with Dynamic Message

Problem Statement:

Design a React functional component that displays a button and a message. Initially, the button is labeled "Show," and no message is visible. Clicking the button should toggle:

  1. The button label between "Show" and "Hide."
  2. A message, such as "Hello, welcome to React state management!" which appears or disappears.

The button should always reflect the current state of the message (e.g., if the message is visible, the button label should be "Hide").
