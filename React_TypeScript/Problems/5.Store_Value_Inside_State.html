Problem Statement

Create a web page that contains an input field and a button. When the user types text into the input field and clicks the button, the text should be stored and displayed on the page.

Requirements:

  • Maintain the current input value while the user types.
  • Store the last submitted value only when the button is clicked.
  • Display the stored value below the input field.
  • Do not update the stored value unless the button is clicked.
