L1 - Building an OTP Input Component with useRef

Problem Statement:

Create a React component for OTP input with four input fields. When the user types a digit in one field, the focus automatically moves to the next. If the user presses backspace, the focus should move to the previous field. Use useRef to manage these behaviors.

Requirements:

  • Implement seamless focus navigation.
  • Use useRef to track input elements.
  • Avoid causing unnecessary re-renders.

Example Interaction:

  1. User types "1" in the first input: Focus shifts to the second input.
  2. User presses backspace in the second input: Focus shifts to the first input.
