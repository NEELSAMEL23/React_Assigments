Problem Statement — Enter and Display Text

Create an HTML page with an input box and a button that displays the entered text on the page when the button is clicked.

Requirements

  • Add an input field for the user to type text.
  • Add a button that, when clicked, shows the text typed by the user.
  • The displayed text should only update when the button is clicked, not while typing.
  • Use only one JavaScript variable to store and display the text.

Functionality Example

  1. User types "Hello" in the input box → Output remains blank until button click.
  2. User clicks the "Show" button → Output updates to "Hello".
  3. User changes input to "World" → Output still shows "Hello".
  4. User clicks the button again → Output updates to "World".
