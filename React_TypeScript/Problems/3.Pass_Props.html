💻 React Challenge

🧩 Problem: Pass Data from Child to Parent

Objective:

Create two React components: a parent and a child. The child component should accept input from the user and send it to the parent component when a button is clicked.

🔧 Requirements:

  • Use useState in both components.
  • The parent should define a function to receive data from the child.
  • Pass this function to the child via props.
  • The child should call the function with its input value.
  • The parent should display the received data.

✅ Example Output:

Child Component:
[ input: "Hello" ] [ Send ]

Parent Component:
Data received from child: Hello

📝 Bonus:

  • Clear the child input after sending.
  • Add simple styling to separate parent and child components.

Files to create:

  • App.js
  • ParentComponent.js
  • ChildComponent.js
