L0 - Using useRef for Input Focus Management

Problem Statement:

Build a React component with an input field and a button. When the button is clicked, the input field should automatically gain focus. Use useRef to implement this behavior.

Detailed Example Input/Output:

  1. Initial State: The input field is not focused.
  2. User clicks the button: The input field gains focus immediately.

Requirements:

  • Use useRef to manage the input focus.
  • Do not cause unnecessary re-renders.
