L0 - Side Effects with useEffect in React

Problem Statement:

Using React and the useEffect hook, create a functional component that fetches and displays a random joke from an external API e.g. https://official-joke-api.appspot.com/random_joke

  1. Display the joke in a card format.
  2. Add a button labeled "Get Another Joke" that, when clicked, fetches a new joke.
  3. Ensure the API call is handled inside useEffect.
