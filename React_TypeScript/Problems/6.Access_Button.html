Problem Statement

Create a React + TypeScript component that uses useRef to directly access and manipulate a DOM element. The component should render a button labeled "Click Me". When the button is clicked, its label should change to "Clicked!" without using React state. Instead, the label update should be done by modifying the DOM element through the useRef reference.

Requirements:

  • Use useRef to store a reference to the button element.
  • On button click, change its text content to "Clicked!".
  • Do not use useState for this functionality.
