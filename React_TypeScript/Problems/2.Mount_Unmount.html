L0 - Understanding the Unmount Phase in React

Problem Statement:

Create a React component that logs a message when it mounts and another message when it unmounts. Use the useEffect hook to handle the cleanup phase. Your component should also have a button to toggle its visibility, simulating the mount and unmount process.

Example Input/Output:

  1. On Mount: Console displays "Component Mounted".
  2. On Unmount: Console displays "Component Unmounted".
  3. User toggles visibility: Component is mounted and unmounted repeatedly.

Requirements:

  • Use useEffect with cleanup logic.
  • Implement a button that toggles the visibility of the component.
