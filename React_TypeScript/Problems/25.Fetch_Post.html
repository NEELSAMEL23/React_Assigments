Overview

This assignment focuses on using Redux for managing the state of an API call in a React application. You will fetch data from a public API and display it in a simple UI using React. The goal is to help you understand how to integrate Redux for state management in basic API operations.

Problem Statement

You need to fetch data from the following public API and display it on the page:

API:

https://jsonplaceholder.typicode.com/posts The application should have the following features:

Redux Store Setup:

Configure a Redux store with an initial state. Create actions and a reducer to handle fetching, success, and failure states.

Fetch Data:

Fetch the posts data from the API and store it in the Redux store.

Display Data:

Show the list of posts on the page, including the title and body of each post.

Loading State:

Display a "Loading..." message while the data is being fetched.

Error Handling:

Show an error message if the API request fails. Code Structure Organize your project files as follows:

├── src/
│ ├── components/
│ │ ├── PostList.jsx
│ ├── redux/
│ │ ├── actionTypes.js
│ │ ├── actions.js
│ │ ├── reducer.js
│ │ └── store.js
│ └── App.js
│ └── index.js

Steps to Solve

Setup Redux:

Create a Redux store using createStore.

Write a reducer to manage API data and loading/error states.

Actions:

Define action types: FETCH_POSTS_REQUEST, FETCH_POSTS_SUCCESS, and FETCH_POSTS_FAILURE. Create action creators for each type.

Fetch Data:

Dispatch a FETCH_POSTS_REQUEST action when the fetch starts.

On successful fetch, dispatch a FETCH_POSTS_SUCCESS action with the fetched data.

On failure, dispatch a FETCH_POSTS_FAILURE action with the error.

Display Posts:

Use useSelector to access the posts from the Redux store.

Display the list of posts in the PostList component.

Handle Loading and Error States:

Show a "Loading..." message while data is being fetched.

Display an error message if the fetch fails.

Example Output

On Load: Display "Loading..."

After Fetch: Display a list of posts with their titles and content.

On Error: Show a message like "Failed to load posts. Please try again."
