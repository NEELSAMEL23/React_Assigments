L2 - Custom Video Player with useRef

Problem Statement:

Design a React video player with "Play," "Pause," and "Restart" controls. Use useRef to manage the video element directly, ensuring minimal re-renders.

Requirements:

  • The "Play" button starts the video.
  • The "Pause" button pauses the video.
  • The "Restart" button resets the video to the beginning.
  • Manage the video element with useRef.

Expected Input/Output:

  1. Initial State: Video is paused at the beginning.
  2. User clicks "Play": Video starts playing.
  3. User clicks "Pause": Video pauses.
  4. User clicks "Restart": Video restarts from the beginning.
